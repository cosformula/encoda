assertDefined

  • assertDefined<T>(value: T | null | undefined): T | never
    • Defined in src/codecs/gdoc/index.ts:147

    Assert that a value is defined.

    In the GDoc Typescript definitions, all properties are optional i.e. potentially undefined. In 44.0.0, all (?) properties were also allowed to be null (https://github.com/googleapis/google-api-nodejs-client/pull/1824/files). In practice though, working with actual documents, many of them are reliably present. So this function provides a null and undefined guard to prevent Typescript complaining while at the same time providing a stack in the case that something is not defined.
