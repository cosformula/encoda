decodeAff

  • decodeAff(aff: Element): Organization
    • Defined in src/codecs/jats/index.ts:983

    Decode a JATS <aff> element into a Stencila Organization node.

    Uses the first <institution> element as the name of the organization. Sometimes there is no <institution> element and <addr-line> is used for name and address combined. So this uses the first address components as the name if necessary.

    When there are multiple <institution> elements this decodes them assuming that they are ordered hierarchically i.e. that the first is a part of the second etc. See https://github.com/stencila/encoda/issues/458

    Parameters

    • aff: Element

    Returns Organization
